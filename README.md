# Proyecto-php

Proyecto final de PHP, realizado por Alejandro Felipe, Alejandro Vico y Julio de la Torre.

## Maquetación del Proyecto

### Flujo de Navegación (User Flow)

1. **Inicio:** El usuario llega y ve productos destacados.
2. **Detalle:** Clic en un producto para ver descripción y precio.
3. **Registro:** Antes de comprar, el usuario crea su cuenta.
4. **Carrito:** Se muestran los productos seleccionados y el total.
5. **Confirmación:** Mensaje de éxito tras "simular" el pedido.
